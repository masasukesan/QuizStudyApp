# departments/ — 10部署のホーム

QuizStudyApp の各 AI 部署はこの配下にある。
各部署のフォルダには CLAUDE.md（部署の方針書）と、その部署の作業ファイル・ドラフト・打ち合わせメモなどを置く。

## 部署一覧

| 部署 | パス | 役割 |
|------|------|------|
| 問題収集部 | `problem-collection/` | 出題構造データの抽出 |
| 問題生成部 | `problem-generation/` | 完全オリジナルの問題作成 |
| 解説制作部 | `explanation/` | 中学生でも分かる解説作成 |
| UI/UX 部 | `ui-ux/` | 画面・体験設計 |
| 開発部 | `development/` | アプリ実装（`/app/` への反映） |
| ゲーミフィケーション部 | `gamification/` | EXP・レベル・バッジ等の設計 |
| 画像・音素材部 | `assets/` | 画像・音声の制作（納品先は `/app/media/`） |
| 広報部 | `marketing/` | アプリ紹介・SNS発信 |
| 経営企画部 | `management/` | 全体統括・品質ゲート |
| 完成物部 | `archive/` | バージョン管理（成果物実体は `/archive/`） |

## 部署作業ファイルと最終成果物の置き場の違い

- **ドラフト・打ち合わせ・草案** → `departments/<部署>/`
- **承認済みの最終成果物**：
  - 教材（問題・解説・構造）→ `curriculum/<教科>/.../`
  - アプリのコード → `app/`
  - 画像・音声 → `app/media/`
  - 確定済みバージョン → `archive/`

各部署が「どこに何を出すか」は、その部署の `CLAUDE.md` の冒頭に明記する。
