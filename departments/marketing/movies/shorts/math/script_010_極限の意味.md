動画タイトル：lim って何を求めているのか

説明文：「x が 0 に近づくとき」の「近づく」とは何か。代入できない場面でも値を確定できる、それが極限の存在意義です。

---

lim という記号
「x が 0 に近づくとき」と読んだことはあると思います

でも「近づく」とはどういう意味でしょうか

x が 0 に「等しい」のとは違います

たとえば
lim(x→0) sinx ÷ x という式があります

x ＝ 0 を代入すると
sin0 ÷ 0 ＝ 0 ÷ 0

これは計算できません

でも x を 0.1、0.01、0.001 と
限りなく 0 に近づけていくと
sinx ÷ x の値は限りなく 1 に近づいていきます

これが極限です

代入はできないけれど
近づけたときの行き先を求める操作です

だから lim(x→0) sinx ÷ x ＝ 1 です

微分の定義式も同じ考え方です

f'(x) ＝ lim(h→0) (f(x＋h) ー f(x)) ÷ h

h ＝ 0 を代入すると 0 ÷ 0 になります
でも「h を限りなく 0 に近づけたとき」の値として
傾きを定義しています

lim は「近づけたときに何に向かうか」を問う記号です

代入できない場面でも極限を使えば値を確定できる
それが lim の存在意義です
