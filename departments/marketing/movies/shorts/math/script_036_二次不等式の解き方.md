二次不等式、どこで詰まっていますか？

コツは「グラフで考える」ことです

例えば x²－3x＋2 ＞ 0 を解きます

まず等号をつけて方程式を解きます
x²－3x＋2 ＝ 0
(x－1)(x－2) ＝ 0
x ＝ 1、2

次にグラフを思い浮かべます
x² の係数が正なので、放物線は上に開いています

グラフが x 軸より上にある範囲を答えます
x ＜ 1、2 ＜ x

もし不等号が ＜ 0 なら
グラフが x 軸より下の範囲
1 ＜ x ＜ 2 になります

符号が正なら「外側」、負なら「内側」
グラフの形をイメージするだけで解けます
