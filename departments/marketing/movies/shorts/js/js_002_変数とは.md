# 台本

変数って何なのかわからない
そんな人に見てほしい動画です

変数はデータを入れる「箱」だと思ってください

たとえばこう書きます
let name = "太郎"

これは「nameという箱に、太郎という文字を入れた」という意味です

箱に名前をつけて
そこにデータを入れる
それだけです

あとから取り出すこともできます
console.log(name)
これで「太郎」と表示されます

箱の中身を入れ替えることもできます
name = "花子"
これで箱の中身が花子に変わります

変数を使えば同じデータを何度も書かなくて済みます
名前を100箇所に書くより
nameを使い回す方がずっと楽ですよね

変数は「データに名前をつける仕組み」です
これだけ覚えれば十分です
