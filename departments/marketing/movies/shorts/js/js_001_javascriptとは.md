# 動画タイトル
HTMLとCSSだけじゃ動かない！JavaScriptの役割【JavaScript基礎#1】

# 説明文
「HTMLとCSSを勉強したけど、JavaScriptって何をするの？」と思ったことはありませんか。この動画では、JavaScriptが「ウェブページを動かす言語」であることを、具体例とコードで60秒で解説します。プログラミング初心者・挫折した方向けにわかりやすく説明しています。

# 台本

HTMLとCSSだけで
ウェブページって作れますよね
でも何か足りないと思いませんか
そう、「動き」です
ボタンを押してもページが変わらない
クリックしても何も起きない
それを解決するのがJavaScriptです
HTMLは骨格を作る言語
CSSは見た目を整える言語
そしてJavaScriptはページを動かす言語
たとえばこういうコードを書くと
document.getElementById("btn").onclick = function() {
  alert("クリックされました")
}
ボタンを押したときにメッセージが出せます
これがJavaScriptの基本です
ウェブページを「動かしたい」と思ったとき
それがJavaScriptの出番ですよ
