# 台本

文字列に変数をつなげるとき
+ だらけになったことはありませんか

実はテンプレートリテラルを使えば
一瞬でスッキリします

結論から言います
バッククォートで囲んで
中に ${} を書くだけです

例えばこう書くと
let name = "太郎"
let msg = "こんにちは、" + name + "さん"

これでも動きますが
+ が増えると読みにくいです

テンプレートリテラルならこうです
let msg = `こんにちは、${name}さん`

バッククォートの中に
${変数名} と書くだけで
自動で埋め込んでくれます

シングルクォートやダブルクォートでは
この書き方はできないので注意してください

バッククォートを使うこと
それだけ覚えれば十分です
