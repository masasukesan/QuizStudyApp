# 台本

同じ処理を何度も書いていませんか

それ、関数を使えば解決します

結論から言います
関数とは処理をひとまとめにして
名前をつけたものです

例えばこう書くと
function greet() {
console.log("こんにちは")
}

これで挨拶する処理に
greetという名前がつきました

使うときはこう呼び出すだけです
greet()

これだけで
中の処理がまとめて実行されます

同じコードをコピペする前に
関数にできないか考えてください

処理をまとめて名前をつける
それが関数です
