# 動画タイトル
functionが矢印に？アロー関数とは【JavaScript基礎】

# 説明文
アロー関数って聞いたことあるけど何が違うの、と感じたことはありませんか。実はfunctionを矢印に置き換えるだけの書き方です。プログラミング初心者・挫折した方向けの60秒解説です。

# 台本

アロー関数って聞いたことありますか
functionを短く書ける方法です

結論から言います
functionを矢印に置き換えるだけです
意味は同じです

普通の関数はこう書きます
function greet() {
console.log こんにちは
}

これをアロー関数にすると
こうなります
const greet = () => {
console.log こんにちは
}

functionという言葉が消えて
かっこの後に矢印がついただけです

呼び出し方は同じです
greet()

まずはこの書き換えだけ覚えてください
それがアロー関数です
