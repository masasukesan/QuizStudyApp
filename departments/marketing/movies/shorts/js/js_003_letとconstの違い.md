# 台本

letとconstって何が違うの？
そう思ったことありませんか

結論はシンプルです
letは書き換えられる変数
constは書き換えられない変数
それだけです

例えばこう書くと
let count = 0
count = 1
これは大丈夫です

でもconstだとこうなります
const name = "太郎"
name = "花子"
これはエラーになります

書き換える予定がないなら
constを使ってください
あとで書き換えたくなったら
letに変えればいいだけです

迷ったらconst
それだけ覚えれば十分です
