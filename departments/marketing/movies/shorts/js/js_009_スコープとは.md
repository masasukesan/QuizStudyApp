# 動画タイトル
変数、どこでも使えると思ってませんか？スコープとは【JavaScript基礎】

# 説明文
関数の中で作った変数を、外で使おうとしてエラーになったことはありませんか。実は変数には「使える範囲」が決まっています。プログラミング初心者・挫折した方向けの60秒解説です。

# 台本

変数の値を外から取り出そうとして
エラーになったことはありませんか

結論から言います
変数には使える範囲があります
これをスコープと言います

例えばこう書くと

function greet() {
let name = "太郎"
console.log(name)
}

greet()
console.log(name)

nameはgreetの中でしか使えません
最後の行はエラーになります

関数の中で作った変数は
関数の外では使えません

外に渡したいときは
returnを使ってください

これがスコープの基本です
