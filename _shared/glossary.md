# 用語集・略称・命名規則（_shared/glossary.md）

QuizStudyApp 全部署が共通で使う用語・略称・命名規則をここに集約する。
表記ゆれが起きたら、まずここを更新してから全部署に展開する。

---

## ■ 階層用語

| 用語 | 英語表記 | 意味 |
|------|---------|------|
| 教科 | subject | 国語・数学・英語・理科・社会の5系統 |
| コース | course | 数学のみで使用。1A / 2B / 3 / C |
| 分野 | section | 数学以外の教科の第一階層（例：english/reading） |
| 大単元 | unit | 例：数学ⅠAの「2次関数」「場合の数」 |
| 小単元 | subunit | 例：「2次関数」配下の「2次方程式」「2次不等式」 |
| 出題パターン | pattern | structure.json 内で抽象化された出題形式の単位 |

## ■ ファイル命名規則

### ● curriculum/ 配下（教材データ）

各小単元フォルダに必ず以下の3ファイルを置く：

```
<小単元>/
├── structure.json      … 問題収集部の出力（出題構造）
├── questions.json      … 問題生成部の出力（30問ストック）
└── explanations.json   … 解説制作部の出力（30問分の解説）
```

### ● 識別子（ID）

```
<教科>-<コース or 分野>-<大単元>-<小単元>-<連番3桁>
例：math-1A-quadratic-functions-quadratic-equations-001
```

### ● フォルダ名

- ケバブケース（`quadratic-equations`）を使用
- 英単語のみ（記号・スペースは不可）

---

## ■ 数式表記ルール（再掲）

- `^2` `^3` 禁止 → **²** **³** を使う
- LaTeX 禁止
- 順列・組み合わせは `nPr` `nCr` 形式（例：`9P2`、`10C3`）
- 解法の俗称（かたまり法・隙間挿入法・補集合法）は使用しない
  → 「AとBを1つのまとまりとして扱う」「残りを先に並べて隙間を数える」「全体から引いて求める」
